# DMMHS School Performance Analytics Dashboard

A comprehensive school performance monitoring dashboard for Diosdado Macapagal Memorial Highschool.

## Features
- Executive Dashboard with real-time KPIs
- Enrollment, Dropout, and Repeater tracking
- SHS Strand management
- Analytics Continuum (Descriptive → Diagnostic → Predictive → Prescriptive)
- Multi-year data (2021-2026)
- Export to Excel/JSON
- Light/Dark theme
